         ```typescript
         }
         ```
       - Create toolbar with sections:
         - Text formatting: Bold, Italic, Underline, Strikethrough
         - Headings: H1, H2, H3 dropdown
         - Alignment: Left, Center, Right, Justify
         - Lists: Bullet, Numbered
         - Media: Image (triggers onImageInsert), YouTube, Table
         - Utilities: Link, Blockquote, Code, Undo, Redo
       - YouTube button: prompt() for URL, then editor.commands.setYoutubeVideo({ src })
       - Table button: insert 3x3 table with editor.commands.insertTable({ rows: 3, cols: 3 })
       - Include BubbleMenu component
       - EditorContent with prose styling (match existing rich-text-editor.tsx)

    2. Create barrel export at `src/components/editor/index.ts`:
       - Export AdvancedEditor with next/dynamic wrapper:
         ```typescript
         import dynamic from 'next/dynamic';

         export const AdvancedEditor = dynamic(
           () => import('./advanced-editor').then(mod => mod.AdvancedEditor),
           {
             ssr: false,
             loading: () => <div className="h-[300px] border rounded-lg animate-pulse bg-gray-100" />
           }
         );
         ```
       - Also export { RichTextEditor } from './rich-text-editor' for backward compatibility

    Why code-split: TipTap + extensions add ~80KB to bundle. Dynamic import keeps initial page load fast.
