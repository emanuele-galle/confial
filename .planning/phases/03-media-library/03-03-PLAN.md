**Focal Point Library (src/lib/focal-point.ts):**
- Export async function `analyzeImageFocalPoint(imageBuffer: Buffer): Promise<{ x: number, y: number, confidence: number }>`
- Use Claude's native vision capability (Anthropic SDK)
- Create message with image attachment (base64 encoded)
- Prompt: "Analyze this image and identify the main focal point (face, subject, or point of interest). Return ONLY a JSON object with x and y as percentages (0-100) from top-left, and confidence (0-1). Example: {\"x\": 45, \"y\": 30, \"confidence\": 0.9}"
- Parse JSON response
- Return { x, y, confidence }
- Handle errors gracefully (return center { x: 50, y: 50, confidence: 0 } on failure)

**API Endpoint (POST /api/media/analyze):**
- Auth required
- Accept: { mediaId: string }
- Fetch image from MinIO
- Call analyzeImageFocalPoint
- Update Media record with focalPointX, focalPointY
- Return focal point coordinates

**FocalPointPicker Component:**
- Props: { media: Media, onChange: (x: number, y: number) => void }
- Display image with crosshair overlay at focal point
- Click on image to manually set focal point
- "Rileva automaticamente" button triggers AI analysis
- Show loading spinner during analysis
- Display confidence badge after detection
- Focal point shown as draggable circle (10px radius)
- Coordinates display below image (x%, y%)
