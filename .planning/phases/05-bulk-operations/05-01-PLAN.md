Extend import endpoint with transactional processing and SSE progress:

**Enhanced POST endpoint:**
- If query param `?execute=true`, perform actual database insert
- Use `prisma.$transaction()` for all-or-nothing processing (BULK-04)
- Stream progress via SSE: `text/event-stream` content type
- Progress events: `{ type: 'progress', current: N, total: M, item: string }`
- Success event: `{ type: 'complete', inserted: N }`
- Error event: `{ type: 'error', message: string, rollback: true }`

**SSE Pattern:**
```typescript
const encoder = new TextEncoder();
const stream = new ReadableStream({
  async start(controller) {
    for (let i = 0; i < rows.length; i++) {
      controller.enqueue(encoder.encode(`data: ${JSON.stringify({type: 'progress', current: i+1, total: rows.length})}\n\n`));
      // Insert row...
    }
    controller.enqueue(encoder.encode(`data: ${JSON.stringify({type: 'complete', inserted: rows.length})}\n\n`));
    controller.close();
  }
});
return new Response(stream, { headers: { 'Content-Type': 'text/event-stream' } });
```

**Transaction handling:**
- Wrap entire insert loop in `prisma.$transaction()`
- On any error, transaction auto-rolls back
- Send error event with rollback confirmation
- Create AuditLog entry: "CSV Import: X news/events importati"
