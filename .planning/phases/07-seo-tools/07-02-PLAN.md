Create template database model and CRUD API:

**Prisma schema (ContentTemplate model):**
```prisma
model ContentTemplate {
  id          String   @id @default(cuid())
  name        String
  description String?
  category    String   @default("generale")
  entityType  String   // "news" | "events"
  content     Json     // Stores title, excerpt, body structure
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
  createdBy   String?
  creator     User?    @relation(fields: [createdBy], references: [id])

  @@index([category])
  @@index([entityType])
}
```

**Content JSON structure:**
```typescript
interface TemplateContent {
  title?: string;
  excerpt?: string;
  body: string; // HTML from TipTap
  category?: string; // For news
  location?: string; // For events
}
```

**GET /api/admin/templates:**
- Query params: `entityType` (optional), `category` (optional)
- Returns: `{ templates: ContentTemplate[] }`
- Ordered by: category ASC, name ASC

**POST /api/admin/templates:**
- Body: `{ name, description, category, entityType, content }`
- Validates: name required, entityType must be "news" or "events"
- Returns: created template

**GET /api/admin/templates/[id]:**
- Returns single template by ID

**PATCH /api/admin/templates/[id]:**
- Update template name, description, category, content

**DELETE /api/admin/templates/[id]:**
- Soft delete or hard delete template

**Run migration:**
```bash
npx prisma db push
```
